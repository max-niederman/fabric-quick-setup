# **Fabric Quick Setup**
### CLI to install Fabric Loader and Popular Mods

[![CLI Demo Image](https://i.postimg.cc/3JnDGLdT/fabric-quick-setup.png)](https://postimg.cc/Vd06QWqG)

## Installation:
### Prerequisites:
- Python 3 & Pip
- Java 1.8 or higher

### Install with Pip:
> pip install fabric-quick-setup
